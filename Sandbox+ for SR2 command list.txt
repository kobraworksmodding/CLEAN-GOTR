Using Sandbox+ for SR2

The SR2 version of Sandbox+ is a bit different than what you might be used to from the SRTT version. This is due to the limited functions in SR2 that are available for capturing input. To use while in game, first press 888 in sequence. This will trigger a code entry message and get things ready for you to input your code. Then, enter one of the following 3 digit codes:

MISC COMMANDS

811 - Toggle Sandbox+ messages displayed in HUD
812 - Toggle if commands also affect coop remote player
813 - Give $100,000
814 - Repair current vehicle
815 - Disable/Enable player weapons (handy for animation commands if you don't want to see the weapons switching)



Examples:
888116 <- Teleport to Docks
888111 < Teleport to Airport


Remember, always enter 888 first to initialize the entry for your actual 3 digit code
